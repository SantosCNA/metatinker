# 元像AI Lab 网站测试计划

## 📋 测试目标

在补充真实资产前，全面验证网站样式设计、交互功能和用户体验，确保所有视觉元素和交互行为符合设计规范。

## 🎯 测试范围

### 1. 视觉样式测试
- [ ] 颜色系统：主色、辅助色、渐变效果
- [ ] 字体系统：中英文字体、字号层级、行高
- [ ] 间距系统：组件间距、页面留白
- [ ] 圆角与阴影：卡片圆角、阴影层次
- [ ] 玻璃拟态效果：背景模糊、透明度
- [ ] 渐变效果：Hero渐变、按钮渐变、文字渐变

### 2. 响应式布局测试
- [ ] 移动端（375px - 767px）
- [ ] 平板端（768px - 1023px）
- [ ] 桌面端（1024px - 1439px）
- [ ] 大屏端（1440px+）
- [ ] 横屏/竖屏切换

### 3. 组件功能测试
- [ ] Navigation：滚动显示、移动端菜单、锚点跳转
- [ ] Hero：粒子动画、按钮交互、能力标签
- [ ] Services：卡片悬停、内容展示
- [ ] Cases：筛选功能、卡片交互
- [ ] About：时间轴、专业领域卡片
- [ ] Blog：分类筛选、文章卡片
- [ ] Contact：联系表单、微信二维码区域
- [ ] Footer：链接、版权信息

### 4. 动画与交互测试
- [ ] 页面滚动动画：淡入、上移效果
- [ ] 悬停效果：按钮、卡片、链接
- [ ] 点击反馈：按钮缩放、状态变化
- [ ] 过渡动画：页面切换、状态变化
- [ ] 性能：动画流畅度、FPS

### 5. 可访问性测试
- [ ] 键盘导航：Tab顺序、焦点状态
- [ ] 屏幕阅读器：语义化标签、ARIA属性
- [ ] 颜色对比度：文字与背景对比
- [ ] 字体大小：最小可读字号
- [ ] 交互元素：按钮大小、点击区域

### 6. 性能测试
- [ ] 首屏加载时间（< 3秒）
- [ ] Lighthouse评分（> 90）
- [ ] 核心Web Vitals：LCP、FID、CLS
- [ ] 资源加载：图片、字体、CSS
- [ ] 代码分割：按需加载

### 7. 浏览器兼容性测试
- [ ] Chrome（最新版）
- [ ] Safari（最新版）
- [ ] Firefox（最新版）
- [ ] Edge（最新版）
- [ ] 移动端Safari（iOS）
- [ ] 移动端Chrome（Android）

## 🛠️ 测试工具与方法

### 开发环境测试
```bash
# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

### 浏览器开发者工具
- Chrome DevTools：性能分析、响应式测试
- Lighthouse：性能、可访问性、SEO评分
- Network：资源加载分析
- Console：错误检查

### 响应式测试工具
- Chrome DevTools Device Mode
- BrowserStack（跨浏览器测试）
- Responsively App（多设备预览）

### 可访问性工具
- WAVE（Web Accessibility Evaluation Tool）
- axe DevTools
- Lighthouse Accessibility Audit
