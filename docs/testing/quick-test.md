# 快速测试指南

## 🚀 立即开始测试

### 1. 启动开发服务器
```bash
npm run dev
```
访问：http://localhost:5173

### 2. 快速视觉检查（5分钟）

#### 颜色系统
- [ ] 打开页面，检查背景是否为深色（#0f172a）
- [ ] 检查主色按钮是否为紫色（#6366f1）
- [ ] 检查强调色是否为青色（#06d6a0）
- [ ] 检查文字颜色对比度是否足够

#### 字体系统
- [ ] 检查标题字体是否为 Space Grotesk
- [ ] 检查中文字体显示是否正常
- [ ] 检查字号层级是否清晰
- [ ] 检查行高是否舒适

#### 布局
- [ ] 检查页面是否居中
- [ ] 检查各区块间距是否合理
- [ ] 检查卡片圆角是否显示
- [ ] 检查阴影效果是否正常

### 3. 响应式快速测试（10分钟）

#### Chrome DevTools 设备模式
1. 按 `F12` 打开开发者工具
2. 点击设备图标（或按 `Ctrl+Shift+M`）
3. 测试以下设备：
   - [ ] iPhone SE (375px)
   - [ ] iPhone 12 Pro (390px)
   - [ ] iPad (768px)
   - [ ] iPad Pro (1024px)
   - [ ] Desktop (1920px)

#### 检查项
- [ ] 移动端导航菜单是否正常
- [ ] 文字是否可读（不会太小）
- [ ] 按钮是否可点击（触摸区域足够）
- [ ] 布局是否适配（无横向滚动）
- [ ] 图片是否响应式

### 4. 交互功能测试（10分钟）

#### 导航
- [ ] 点击导航链接，页面是否平滑滚动
- [ ] 滚动页面，导航栏是否固定
- [ ] 移动端点击菜单按钮，菜单是否展开
- [ ] 移动端点击菜单项，菜单是否关闭

#### 按钮
- [ ] 悬停按钮，是否有视觉反馈
- [ ] 点击按钮，是否有点击动画
- [ ] 按钮文字是否清晰可读

#### 卡片
- [ ] 悬停卡片，是否有上浮效果
- [ ] 卡片内容是否完整显示
- [ ] 卡片间距是否合理

#### 动画
- [ ] 滚动页面，元素是否淡入
- [ ] 动画是否流畅（无卡顿）
- [ ] 动画时长是否合适

### 5. 性能快速检查（5分钟）

#### Lighthouse 测试
1. 打开 Chrome DevTools
2. 切换到 Lighthouse 标签
3. 选择 "Performance" 和 "Accessibility"
4. 点击 "Analyze page load"
5. 查看评分：
   - [ ] Performance > 80
   - [ ] Accessibility > 80

#### 网络检查
1. 打开 Network 标签
2. 刷新页面（Ctrl+R）
3. 检查：
   - [ ] 页面加载时间 < 3s
   - [ ] 资源大小合理
   - [ ] 无404错误

### 6. 浏览器兼容性快速测试（15分钟）

#### 桌面浏览器
- [ ] Chrome（最新版）
- [ ] Safari（最新版）
- [ ] Firefox（最新版）
- [ ] Edge（最新版）

#### 移动浏览器
- [ ] iOS Safari
- [ ] Android Chrome

#### 检查项
- [ ] 样式是否一致
- [ ] 功能是否正常
- [ ] 是否有错误提示
